= DogFormatter
date format gem

= install

```
$ bundle isntall dog_formatter
```

or 

```
gem 'dog_formatter'
```

= usage

```
1 require 'dog_formatter/model'
2
3 class User < ActiveRecord::Base
4 end
```

```
user.created_at_ymd
=> "2016/11/04"
user.created_at_ymd_ja
=> "2016年11月04日"
user.created_at_ym
=> "2016/11"
user.created_at_ym_ja
=> "2016年11月"
```
